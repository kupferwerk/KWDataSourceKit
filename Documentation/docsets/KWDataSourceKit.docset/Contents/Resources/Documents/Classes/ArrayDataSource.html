ArrayDataSource

open class ArrayDataSource<CellType: Reusable, ItemType>: BaseDataSource<CellType, ItemType>

ArrayDataSource is a lightweight DataSource that can be used as a UITableViewDataSource or UICollectionViewDataSource. ArrayDataSource always uses a single section. To change the items served by the dataSource set the items property.

Note

If you need multiple sections, use SectionedDataSource instead
  • items

    The items served to a UITableView or UICollectionView by the dataSource

    Note
