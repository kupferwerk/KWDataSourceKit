Reusable

public protocol Reusable

A protocol that marks type as beeing reusable Every type implementing Reusable will receive a computed static property reuseId that can be used to identifiy the type. Reusable also provides a default implementation for reuseId, which returns the type’s name, i.e. for the type SomeTableViewCell the string SomeTableViewCell will be returned. - Note: Overriding reuseId is supported for type’s that do not match the above description.
